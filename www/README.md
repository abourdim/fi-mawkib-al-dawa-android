# في موكب الدعوة — In the Procession of Dawah

Interactive trilingual web app based on **"Fi Mawkib al-Dawa"** by Sheikh Mohammed al-Ghazali (1917-1996).

## About the Book

A collection of articles written before and after the 1952 Egyptian revolution. Themes include social justice in Islam, fighting despotism, women's rights, critique of corrupt governance, the Palestine cause, and confronting deviant religiosity. Al-Ghazali writes as a passionate reformer exposing hypocrisy and calling for authentic Islam.

## Features

- **Trilingual**: Arabic, English, French
- **20 Dawah Lessons**: Key themes from the book with Quran verses and reflections
- **Historical Timeline**: Journey through Islamic preaching history from 610 CE to 1996
- **Quiz**: "Who Wants to Be a Preacher?" with 15 questions and lifelines (50/50, Guidance, Ask the Sheikh)
- **XP & Badges**: Gamification system with 5 badge levels
- **3 Themes**: Ocean (light blue), Night (dark), Sky (white blue)
- **Narrator**: Full audiobook narrator with karaoke, speed/pitch control, sleep timer, duo reading
- **PWA**: Installable as a mobile app
- **Responsive**: Mobile-first design

## Tech

- Vanilla HTML/CSS/JS — no frameworks
- Web Speech API for narrator
- Web Audio API for sound effects
- LocalStorage for persistence
- Genius library for particle effects
